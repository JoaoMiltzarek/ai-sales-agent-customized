# AGENTS.md

## Objetivo do projeto
Este projeto é um MVP chamado `ai-sales-agent-customized`.

Ele deve gerar respostas comerciais personalizadas com base em:
- tipo de negócio
- nome da empresa
- tom de voz
- objetivo da resposta
- mensagem do cliente

## Regras obrigatórias
- Trabalhar em etapas pequenas.
- Não refatorar fora do escopo pedido.
- Não adicionar bibliotecas sem necessidade.
- Não criar autenticação, banco de dados, dashboard, CRM, WhatsApp ou deploy complexo agora.
- Priorizar clareza, simplicidade e organização para um leigo entender.
- Antes de alterar arquivos, listar quais arquivos serão criados ou modificados.
- Sempre explicar como testar localmente ao final de cada etapa.
- Usar nomes de arquivos e funções claros.
- Manter o projeto simples, bonito e funcional.

## Stack desejada
- Next.js
- TypeScript
- Tailwind CSS

## Escopo atual
MVP sem banco de dados e sem autenticação.
