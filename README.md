# ai-sales-agent-customized

MVP simples para gerar respostas comerciais personalizadas com base em:

- tipo de negocio
- nome da empresa
- tom de voz
- objetivo da resposta
- mensagem do cliente

## Stack

- Next.js
- TypeScript
- Tailwind CSS

## Como rodar localmente

```bash
npm install
npm run dev
```

Abra [http://localhost:3000](http://localhost:3000) no navegador.

## Etapa 1

Esta primeira etapa entrega:

- base do projeto com Next.js
- TypeScript configurado
- Tailwind CSS configurado
